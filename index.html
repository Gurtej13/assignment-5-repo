How to Apply CSS to HTML

To include an external CSS file in an HTML file, use the <link> tag inside the <head> section, like this: 

<link rel="stylesheet" href="styles.css">

How to Use JavaScript in a Webpage

To link to an external JavaScript file, use the <script> tag with the src attribute, like this: 

<script src="script.js"></script>
To write JavaScript code within an HTML file, use the <script> tag directly in the <body> or <head> section, like this: 
<script>
    // JavaScript code here
</script>
